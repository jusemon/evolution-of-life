# Kirby Clone

A js13k learning purpouse game using Kirby's Dream Land as base

## Libraries Used

- [Kontra](https://straker.github.io/kontra/)
- [TinyMusic](https://github.com/kevincennis/TinyMusic)
- [Pixel Font](https://github.com/PaulBGD/PixelFont)
